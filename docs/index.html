📲 Installation Instructions

  1. Tap the download button above on your Android device
  2. When the download completes, tap on the APK file
  3. If prompted, enable installation from unknown sources in your device settings
  4. Tap "Install" and wait for the installation to complete
  5. Tap "Open" to launch the app and start using it

📋 System Requirements

  • Android 7.0 or higher
  • Internet connection for syncing with Salesforce
  • Camera access for photo capture and barcode scanning
  • Location services for GPS coordinates
  • Storage permission for saving photos locally
